## インストール

1. リポジトリをクローン

```bash
git clone https://github.com/daizyoo/shogi_chess
```

2. ディレクトリへ移動

```bash
cd shogi-chess
```

3. 依存関係をインストール

```bash
bun install
```

## 実行方法

コンソールで実行

```bash
bun run src/core/main.ts
```
